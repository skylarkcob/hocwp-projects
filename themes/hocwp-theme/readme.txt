HocWP Theme is the core package to develop themes for WordPress, made by hocwp.net.

== Description ==

HocWP is the code used for web programming with WordPress with simple, easy and convenient, you can only use it to code WordPress theme.

If you have any questions or you found a bug, please contact the development team [here](http://hocwp.net/).

== Installation ==

= Minimum Requirements =

* WordPress 4.4 or greater
* PHP version 5.4 or greater
* MySQL version 5.0 or greater

= Notice =

This code is only used to develop themes for WordPress, you must get HocWP core for it works.
